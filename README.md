# Quick notes on Git

* __Get Repo__

 `git clone URL` (Substitute `URL` with repo URL)
  
* __Upload__
  
  `git add FILENAME`
  
  `git commit` or `git commit -m'MESSAGE'`
  
  `git push`
  
* __Sync__ 

  `git pull`
  
  Or
  
  `git fetch --all` followed with `git reset --hard origin/master` / `git reset --hard origin/<branch_name>`
